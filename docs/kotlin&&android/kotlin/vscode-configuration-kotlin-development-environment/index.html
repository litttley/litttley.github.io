vscode配置kotlin开发环境

依赖环境安装​

kotlin依赖java环境,所以要先安装好jdk,并配置好java环境变量.这里不做介绍. oracle 官网下载地址

kotlin安装​

kotlin github 下载地址,选择最新版本下载kotlin-compiler-1.7.20.zip(当前最新版本为1.7.20)

将解压的内容移动一个固定目录并配置path环境变量

新开一个cmd窗口输入kotlin -version 

litttley>kotlin -version
Kotlin version 1.7.20-release-201 (JRE 19.0.1+10-21)

显示版本证明成功.

配置vscode​

1.在Vscode中安装插件Kotlin Language 和Code Runner, 安装完后重启Vscode 2.在vscode 用户设置中添加

"code-runner.runInTerminal": true,
"terminal.integrated.defaultProfile.windows": "Command Prompt",

3.在带有main函数入口kt文件中右健点击run code运行,默认快捷键为ctrl+alt+N

4.或使用命令行方式如下

 kotlinc Main.kt -include-runtime -d Main.jar && java -jar Main.jar

或分开执行

 kotlinc Main.kt -include-runtime -d Main.jar  
 java -jar Main.jar
