giteer仓库代合并到github代码库

giteer代码库同步到github上云​

1.下载要同步到github的代码库 git clone git@gitee.com:xxxx/xxxx.git

2.添加github远程库地址 git remote add origin2 git@github.com:xxxx/xxxx.git

注由于giteer代码库远程名为origin,所以这里命名远程库名称为origin2

3.拉取gitub代码 git fetch origin2

4.通过远程分支创建本地分支重命名为main2

git checkout -b main2 origin2/main 注main2是基于远程分支origin2/main创建的本地分支 4.切换到分支main2下 合并 main分支 git merge main

5.推送main2分支到origin2/main git pull

总节​

具体步骤是添加要推送代码库的远程地址,git add remote origin2 xxxx,然后拉取该远程地址的代码git fetch origin2,然后创建该远程分支的本地分支main2(即我们要开发或要合并的分支),然后再合并main分支 注main2是远程分支本地分支,即我们开发代码时分支。
