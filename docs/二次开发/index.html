drawio 二次开发

装ant​

https://ant.apache.org/bindownload.cgi

编译​

cd etc/build

ant

注我们只是使用ant将drawo编译成压缩版这个功能,不使用src/mian/java里的功能,因为我们只使用前端的代码

ant 编译完文件会替换webapp/js里的压缩代码（*.min.js）,这里的代码在浏览没有设置dev=1参数时使用

开发流程说明​

在开发过程中我们使用(vscode)里live server功能直接将drawio的index.html页面打开,然后在地址栏添加？dev=1这个参数如

地址栏参数说明​

``http://127.0.0.1:8080/src/main/webapp/index.html?dev=1&lang=zh&browser=1&gapi=0&db=0&od=0&tr=0&gh=0&gl=0&device=1&mode=browser&libs=flowchart;general&splash=0&title=Untitled%20Diagram.drawio&create=http%3A%2F%2F127.0.0.1%3A8081%2Ftemplates%2Fflowcharts%2Fdata_flow_1.xml&sync=manual#LUntitled%20Diagram.drawio```

dev:1 开发时使用,将加载非压缩的js
lang:zh 默认语言
browser=0/1: Disables local storage as a storage location (0) or shows the browser option in the storage dialog (1).本地和浏览器存储
gapi=0: Disables the Google integration.google服务
db=0: Disables the Dropbox integration.Dropbox服务
od=0: Disables the OneDrive integration.OneDrive服务
tr=0: Disables the Trello integration.Trello服务
gh=0: Disables the Github integration.Github服务
gl=0: Disables the GitLab integration.GitLab服务
drive=0: Simulates app.diagrams.net regardless of the domain name (uses the old app ID).
mode=[google|onedrive|github|dropobox|device|browser]: Switches to the specified mode. 配合browser=1时使用
title: 使用模板的标题
create：使用模板的url地址
sync
splash=0: Does not show the splash screen.启动画面

splash=1时将弹出

mode=browser

以上配置在文件webapp/js/PreConfig.js里可以代码配置
