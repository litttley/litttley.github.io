浏览器调式之js文件替换

安装 Requestly及使用​

Requestly 浏览器扩展（适用于 Firefox 和 Chrome）。它允许您拦截和重定向浏览器的网络请求，并将请求的资源文件（如 JS 文件）替换为从您自己机器的本地主机或任​​何其他远程服务器提供的资产。

打开open app 配置（可能需要fanqiang）

点击新建规则（new rule）t在这里插入图片描述 选择Redirect Request->创建 下

if request 输入要拦截的js文件请url Redirect to 输入本地服务器路径，本地服务器使用 Live Server vscode扩展

安装 Live Server vscode​

修改配置

  "liveServer.settings.port": 8080, //设置本地服务的端口号
"liveServer.settings.root": "/", //设置根目录，也就是打开的文件会在该目录下找
"liveServer.settings.CustomBrowser": "chrome", //设置默认打开的浏览器
"liveServer.settings.AdvanceCustomBrowserCmdLine": "chrome --incognito --remote-debugging-port=9222",

"liveServer.settings.NoBrowser": false,
"liveServer.settings.ignoredFiles": [//设置忽略的文件
    ".vscode/**",
]

测试

意插件 一定要点击处于激活状态
