blender学习笔记

快捷键​

G 抓取

  • G +X 锁定x轴移动
  • G +Y 锁定Y轴移动
  • G +Z 锁定Z轴移动
  • G +MMB 锁定距离鼠标最近轴线移动 .MMB鼠标中键即middle mouse buttonMMB 旋转视图(按住鼠标中键移动) shift + MMB+ 平移视图 ctrl +MMB+ 缩放视图 shift+space 打开操作面版

~ 视图快捷面板

  • Top 8 顶视图(数字小键盘:8)
  • Font 7 前视图(数字小键盘:7)
  • Back 9后视图(数字小键盘:9)
  • Left 4左视图(数字小键盘:4)
  • Right 6右视图(数字小键盘:6)
  • View Camera 1相机视图(数字小键盘:1)
  • View Selected 3选中最大化显示(数字小键盘:3)
  • Bottom 2顶视图(数字小键盘:8)

数字键盘5用户透视图与用户正交视图切换; Alt+ MMB+ 移动鼠标 视图切换 delete删除 x 确认删除

shift+A 添加物体菜单

F9 打开最近一次操作面板 例如最后一次操作是添加物体F9调出如下面版 最近一次操作是移动F9调出如下面版 注：blender 添加物体参数只能设置一次,完成首次设置且未执行其他操作之前如移动可通过F9再次调出设置面板设置;

N面板

数字键盘

  • . 最大化显示选中物体(视图)
  • + 放大选中物体(视图)
  • - 缩小选中物体(视图)

tab 切换 Object Mode对象模式 与Edit mode 编辑模式

ctrl + tab 调出模式切换快捷菜单如下图

O 衰减编辑 注意:衰减编辑时滚鼠标中键调整笔刷大小

也可通过F9调出移动面板设置笔刷大小

alt +z 透视模式

shift+ d 复制

选中物体+F2 重命名

z 显示模式快捷菜单

捕捉 注意:开启捕捉要配合ctrl使用才能移动

H隐藏选中物体

alt+h 显示隐藏物体
