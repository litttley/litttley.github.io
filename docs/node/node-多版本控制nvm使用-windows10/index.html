node-多版本控制nvm使用-windows10

安装​

windows版本nvm gitub https://github.com/coreybutler/nvm-windows 下载地址 https://github.com/coreybutler/nvm-windows/releases

1.双击解压后的文件nvm-set

2.选择nodeks安装路径(补图，目录为D:\soft\vnm\nodejs,注文件夹名不要有空格)

3.配置阿里源 在nvm安装目录中的settings.txt尾部追加如下代码，换成阿里的镜像

   node_mirror: https://npm.taobao.org/mirrors/node/
    npm_mirror: https://npm.taobao.org/mirrors/npm/

或命令行 阿里

 # 设置npm_mirror:
    nvm npm_mirror https://npmmirror.com/mirrors/npm/
     
    # 设置node_mirror:
    nvm node_mirror https://npmmirror.com/mirrors/node/

腾迅


 # 设置npm_mirror:
    nvm npm_mirror http://mirrors.cloud.tencent.com/npm/
     
    # 设置node_mirror:
    nvm node_mirror http://mirrors.cloud.tencent.com/nodejs-release/

常用命令

nvm install 10.16.2 //安装
nvm use 10.16.2
npm install -g yarn //安装yarn
